Example 04

Example 04 is a set of exampels with a unuiqe feature:
  • Single Web Worker - Run the code in the background
  • Multi Web Worker - Multithreading
  • Indexed DB - Save and Reload scene
  • Geometry Optimization - Merge meshes into a single mesh
A combination of these features can be found here:
  • All Together

NOTE: Each example (in this folder) needs additional files in order to work.
Please run `npm run build-examples` to generate these build files.
